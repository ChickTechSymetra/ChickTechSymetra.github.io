10Next, we'll copy some files "starter" files. For now, please copy the files I sent you in email into the folder that your repository is pointing at. Hopefully there's a spot in setup that we can simply point the repository to the folder that ChickTech will have set up ahead of time.

11Now, let's open Komodo Edit. There should be an icon on your desktop. We'll use this as a handy editor, but you could use any editor.


12Let's edit your home page, called Index.html. Open the file in Komodo.



13Change the green text inside the header to whatever you like, then save the file.


14In Komodo, you can preview your file by clicking the Preview icon [add image]

15Now, let's switch over to the GitHub Desktop and publish your web page to the internet:

  • Enter a 'Summary' for your change, such as "Header text added"
  • Click Commit to master
  • Click Push origin

Give it a minute, and...your page is now out there on the web!

Check it out by using a browser, and going to https://username.github.io
