Text Input Field

Creating an input field is simple:
<input type="text" id="NickName" autofocus />

In this example, the word 'autofocus' tells the browser that this field should be ready for input when someone comes the page.

Multi-line Input Field

For our larger box, we use a textarea tag and tell the browser to create multiple 'rows'.

<textarea id="Quote" rows="7" cols="50"></textarea>

Radio Buttons

Radio buttons are used when there are multiple options, but only one can be chosen. Each button must have the same 'name' so they stay grouped together.

<input type="radio" name="Snack" value="chips" />Chips
<input type="radio" name="Snack" value="chocolate" />Chocolate
<input type="radio" name="Snack" value="cake" />Cake & Pie
<input type="radio" name="Snack" value="vegetables" />Vegetables

Submit Button and Message

A button is simple to add to a page: <button id="ThatMe">That's Me</button>

Next, we need to add some JavaScript inside our <script> tags:

$(function () {
 //This function will show a message, summarizing the data
 $("#ThatsMeButton").on("click", function () {
  var s = "I am ";
  s = s + $("#NickName").val() + ". ";
  s = s + "I'm lucky with number " + $("#LuckyNumber").val() + ". ";
  s = s + "My favorite color is " + $("#FavColor").val() + ". ";
  s = s + "Feed me " + $('input[name=Snack]:checked').val() + "! ";
  s = s + "Here's what I think is cool: " + $("#Quote").val();

  alert(s); //An 'alert' creates the message
});

This will put all the data into a variable called 's'. Then we'll use the 'alert' command in Javascript to show the message.
