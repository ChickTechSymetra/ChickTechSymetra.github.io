Setting up on a different machine

This workshop was meant to be used with ChickTech laptops, which already have some files loaded. But you can do this on any PC or laptop, if you first follow these steps.

A Install Git on your machine. Go to https://git-scm.com/downloads, and download for your operating system. Once the install exe is downloaded, click it to install the Git program.

B Create a folder on your machine where we'll copy some code files. Our example uses C:\WorkshopFiles, but yours can be anything.


C Copy some starter files on to your machine. Click each link below for a page, and save it to the folder you created. To save the page, right-click then select Save as.... Be sure to use the exact names you see below to save your files.

FileSave it as...
Navigation Bar Save as nav.html
Home page Save as index.html
Cool Stuff page Save as coolStuff.html
Other Links page Save as otherLinks.html
Stylesheet Save as myStyle.css
Git Commands Save as SetUpGit.cmd


Now you're ready to go back to Step 7 of the original instructions.
