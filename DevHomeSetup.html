Setting up on a different machine

This workshop was meant to be used with ChickTech laptops, which already have some files loaded. But you can do this on any PC or laptop, if you first follow these steps.

A Go to a Command Prompt (click the Windows icon and type Command, and select Command Prompt.) Once it opens, it will look something like this:



B Check if the Git program has already been installed on this machine.

Inside the Command prompt, type git If...then...

C Install Git...

D Create a folder on your machine to put your code files. Our example uses C:\WorkshopFiles, but yours can be anything.

E Copy some starter files on to your machine. Click each link below for a page, and save it to the folder you created. To save the page, hold Control and click S, then save as the name shown below.

Navigation Bar Save as nav.html
